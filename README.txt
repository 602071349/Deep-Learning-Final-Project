Analysis of Different Transfer Learning Approaches on Natural Scenes Classification

In this project, our code is written as a Jupyter notebook: code.ipynb. Please install Jupyter notebook by following the tutorial:
https://jupyter.org/install

Our code also uses the following Python libraries: Tensorflow, Keras, Matplotlib, numpy, sklearn. Please install those packages before running the code.

The dataset we used is the Intel Natural Scene Classification dataset available at: 
https://www.kaggle.com/puneet6060/intel-image-classification
